How to install carefully

To install software with Pacman, you have to agree to trust caches. Even with this trust, however, Pacman gives you a number of tools for being careful about installing a package from a remote site. If you are thinking about installing BU:Python for example, you can first

  • Browse the package with 
    % pacman -l BU:Python
  • Fetch the package before installing it with 
    % pacman -fetch BU:Python
  • Examine all the shell commands that might be executed with 
    % pacman -domain -dom all | grep shell
  • You can extract and examine the Pacman source codes before installation with 
    % pacman -extract-sources .
  • You can extract and examine all of the downloads before installation with 
    % pacman -extract-downloads .
  • Step through the installation stopping at every shell command with 
    % pacman -install -ask shell
    or with % pacman -install -ask action to stop at each Pacman action.
You can also use caches accessible via Globus or SSH to take advantage of standard SSH or Globus authentication mechanisms.
